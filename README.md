# Tic Tac Toe Game 🎮

A simple Tic Tac Toe game developed using Python and Tkinter.

## Features
- Player vs Computer
- Interactive GUI
- Automatic winner checking
- Restart game option

## Technologies Used
- Python
- Tkinter

## How to Run
1. Open the project in VS Code
2. Run the file:
   ```bash
   python tic_tac_toe.py
